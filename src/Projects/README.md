Forumla 1 Smart Library 
This project utilizes java and javaFx (GUI) to create a Formula 1 Season Tracker. It has drivers info, team info, race results, point tracking, current poll standing(updates after each race)

Project Structure
src
    Projects
        Main.java
        manageRace.java
        Driver.java
        Team.java
        raceResults.java
        README.md
    data/ 
        drivers.csv
        race.csv
        team.csv

All the files are in package src.Projects
Features
Driver.java
* Stores the drivers informationa and statistics and updates after each race. Auto sorting so when you want to list the drivers it will print int order whoever has the most points to the least points.
Team.java
* Stores the team information 
manageRace.java
* prints race results and information on drivers and teams
raceResults 
* keeps track of race details after each entry/update
Main
* use this file to run program
GuiMain
* main file for GUI
